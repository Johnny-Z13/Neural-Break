### Movement Mastery

- Gradual acceleration

**Rotation:**
- Ship faces movement direction

### Weapon Heat Management

**Heat System:**
- Weapon generates heat when firing
- Overheating pauses firing briefly
- Heat dissipates over time

**Optimal Firing:**
- Burst fire for sustained DPS
- Watch heat bar on HUD
- Upgraded weapons heat slower
- Plan firing around cooldowns

## Troubleshooting

### Controller Not Working

**Check:**
1. Controller connected before starting game
2. Browser supports Gamepad API (Chrome, Firefox, Edge)
3. Try unplugging and reconnecting
4. Try different USB port
5. Check browser console for errors

**Supported Controllers:**
- ‚úÖ Xbox One / Series controllers
- ‚úÖ PlayStation 4 / 5 controllers
- ‚úÖ Most generic USB controllers
- ‚ö†Ô∏è Some Bluetooth controllers may have issues

### Keyboard Issues

**If keys don't respond:**
1. Click on game window to focus
2. Check if browser has key bindings (F11, Ctrl+W, etc.)
3. Try different browser
4. Disable browser extensions

### Input Lag

**If controls feel sluggish:**
1. Close other browser tabs
2. Check system resources
3. Disable browser extensions
4. Try lower graphics settings (future feature)
